Conditionals

Python uses indentation (not curly braces) to define code blocks:

python age = 18

if age >= 18:
  print("You can vote!")
else:
  print("Wait until you're 18")

# elif (else if)
if score >= 90:
  grade = "A"
elif score >= 80:
  grade = "B"
else:
  grade = "C"

Loops

Python has two main loop types: for and while:

python # For loop with range
for i in range(5): # 0 to 4
  print(i)

# While loop
count = 0
while count < 3:
  print("Loop", count)
  count += 1 # Important!
